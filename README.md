# QRify

A modern, fast, and customizable QR Code Generator built with **Next.js 16**, **TypeScript**, and **Tailwind CSS**.

Generate beautiful QR codes for websites, text, emails, phone numbers, WiFi, SMS, contacts, and more—all completely free.

---

## ✨ Features

- 🌐 URL QR Code
- 📝 Text QR Code
- 📧 Email QR Code
- 📞 Phone QR Code
- 💬 SMS QR Code
- 📶 WiFi QR Code
- 👤 vCard QR Code

### QR Customization

- 🎨 Foreground Color
- 🎨 Background Color
- 📏 QR Size Control
- 🖼 Logo Upload
- ⚡ Live QR Preview
- 📥 Download as PNG
- 📥 Download as SVG

---

## 📦 Installation

Clone the repository

```bash
git clone https://github.com/Bipin005/QRify.git
```

Move into the project

```bash
cd QRify
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Visit

```
http://localhost:3000
```

---

## 🏗 Build

```bash
npm run build
```

Production Preview

```bash
npm start
```
